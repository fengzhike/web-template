## web-template 前端模板项目

#### 项目简介

该模板项目为了 快速创建一致的前端项目。

项目采用前后端分离架构，登录模块采用统一 xxx 登录平台。兼容目前主流浏览器

技术栈 webpack + es6 + vue + vue-router + vuex。
